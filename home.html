O elemento HTML <blockquote>(ou elemento HTML de citação de bloco) indica que o texto incluido é um longa citação.Normalmente, este é processado visualmente pelo recuo(ver Notas sobre como mudá-lo). A URL para a fonte da citação pode ser dada usando o atributo cite, enquanto uma representação de texto da fonte pode ser dada usando o <cite> elemento.
